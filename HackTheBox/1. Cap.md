# Case Study

## Introduction

### Cap Lab

#### Summary 
The following is based on a fictional company:

The Cap Lab is a new monitoring web-based application, with the purpose to monitor any activities and traffics of every users in it. The manager of IT decided to do a vulnerability assessment and penetration testing before the release of the lab.

### Objectives

1. Assessing vulnerabilities in the web application
2. Test Endpoints
3. Escalate privilege persistency
4. Gain a complete access of system

#### In-Scope Assets
